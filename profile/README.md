## 🎨 About Us

We're **Kernow Creative Group** – a digital creative media solutions studio specializing in web design, brand identity, and digital strategy. Based in Cornwall, we transform ambitious ideas into remarkable digital realities.

### 💡 What We Do

- **🌐 Website Development** - Custom web applications and responsive websites built with modern technologies
- **☁️ Web Hosting & Server Administration** - Reliable hosting solutions and expert server management
- **🏠 Home Tech Solutions** - Smart home integration and technical support
- **🎙️ Radio Production** - Professional audio production and broadcasting services
- **🎨 Brand Identity & Design** - Creative branding that makes you stand out
